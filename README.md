# Projeto Banco (Python)

Esse aqui é um projeto simples de um sistema bancário feito em Python puro e usando POO.  
É só para demonstrar o uso de classes, objetos, algumas boas práticas e entendimento da linguagem. 

---

## O que tem aqui?

- Cadastro de cliente  
- Criação de conta bancária  
- Diversos métodos da classe banco para modificar, deletar, adicionar, visualizar clientes
- E de novo, é apenas um projeto simples no qual eu utilizo somente Python, sem bibliotecas externas

---

## Como ele funciona?

 Um usuário consegue criar clientes, no qual é possível adicionar ao banco e posteriormente adicionar em um arquivo csv,
também sendo possível manipular esse objeto cliente de todas as formas possíveis.
 Caso queira testar o programa, é só rodar no terminal:

```bash
python main.py
```

OBS: Você precisa estar utilizando o Python pelo menos na versão 3.x
